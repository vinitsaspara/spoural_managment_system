# 💍 Spoural Management System (MERN Stack)

This is a full-stack web application built using the **MERN stack (MongoDB, Express.js, React.js, Node.js)**. The Spoural Management System is designed to manage and maintain Spoural-related records in a university environment, with secure role-based access for different users such as **Admin**, **Student**, **Faculty**, and **StudentCoordinator**.

---

## 🌟 Features

- ✅ **Secure Authentication** (JWT-based)
- 🔑 **Role-Based Authorization** (Admin, Faculty, Student, StudentCoordinator)
- 📋 **Manage Spouse Records** (Add, Update, View, Delete)
- 🖥️ **Clean and Responsive UI** with Tailwind CSS and ShadCN UI
- 📦 **Modular Backend Structure** (Controllers, Routes, Models)
- 🌐 **RESTful APIs** for smooth frontend-backend communication

---

## 🔧 Technologies Used

### Frontend
- React.js
- Redux Toolkit
- Tailwind CSS
- ShadCN UI

### Backend
- Node.js
- Express.js
- MongoDB & Mongoose
- JWT (Authentication)
- Bcrypt (Password Hashing)
- CORS, Dotenv

## ⚙️ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/vinitsaspara/spoural_managment_system.git
cd spoural_managment_system
cd backend
npm install
npm run dev

cd ../frontend
npm install
npm run dev

// configure .env file
```
