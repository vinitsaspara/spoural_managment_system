# Spoural Management System

A **web-based cultural event management platform** designed for universities and institutions to manage, schedule, and track cultural activities. The platform connects students, student coordinators, faculty members, and administrators in a **role-based system** for efficient event organization.

---

## 🚀 Features

- **Role-Based Authentication & Authorization**
  - Students, Student Coordinators, Faculty, and Admin roles
- **Event Listings**
  - View all upcoming and ongoing cultural events
- **Event Registration**
  - Students can register for multiple events
- **Participant Management**
  - Coordinators approve and manage participants
- **Event Scheduling**
  - Assign dates, times, and venues
- **Result Management**
  - Publish winners and rankings
- **Notifications**
  - Real-time event updates and alerts
- **Reports & Analytics**
  - Track participation and event performance

---

## 🛠 Tech Stack

**Frontend:**
- React.js (Vite)
- Tailwind CSS
- Axios

**Backend:**
- Node.js
- Express.js (ECMAScript Modules)
- MongoDB

**Architecture:**
- RESTful API
- Role-based access control
- Modular backend structure

## ⚙️ Installation & Setup

1. **Clone the repository**
```bash
git clone https://github.com/vinitsaspara/spoural_managment_system.git
cd spoural_managment_system
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

📌 Use Case Example
Student views upcoming events and registers for a music competition.
Student Coordinator approves the participant and schedules the event.
Faculty monitors event progress and validates results.
Admin oversees the entire process and publishes final winners.
